# TutorUsMath

**Solucionador de ecuaciones cuadráticas**  
App educativa hecha con [Streamlit](https://streamlit.io/) y [SymPy](https://www.sympy.org/) para enseñar, visualizar y resolver ecuaciones cuadráticas, mostrando soluciones, discriminante, gráfica interactiva y el vértice.

---

## ¿Qué hace esta app?

- **Resuelve ecuaciones cuadráticas** de la forma `ax² + bx + c = 0`
- **Explica paso a paso**, incluyendo discriminante y tipo/ cantidad de soluciones.
- **Muestra los coeficientes y la fórmula general.**
- **Visualiza la gráfica interactiva** con los puntos de corte (raíces) y el vértice resaltado.
- **Creada para apoyar la docencia** en matemáticas de nivel medio y superior.

---

## Uso rápido 📚

1. **Instala dependencias:**

   ```
   pip install -r requirements.txt
   ```

2. **Ejecuta la app local:**

   ```
   streamlit run app.py
   ```

3. **Abre en tu navegador** la dirección que se muestra (¡generalmente [http://localhost:8501](http://localhost:8501)!)
